PriTama GO QR Codes List
========================

This page lists all the QR codes that PriTama GO recognizes and can display for
Kiratto Pri☆Chan. Click on the QR code to open it by itself.

Names primarily from the official [secret codes page](https://prichan.jp/toy/pritamago/detail/secret.html).
Graphics directly exported from firmware data. Content © T-ARTS / syn Sophia /
TV Tokyo / PCH3 Production Committee
